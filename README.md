# Учебный проект: интернет-магазин на Django

Этот репозиторий — **учебный**. Код и структура проекта были сделаны для практики и обучения: изучение моделей Django, представлений (views), шаблонов, маршрутизации, а также корзины.

## Что внутри

- `main/` — каталог товаров/категорий и страницы карточек товара.
- `cart/` — логика корзины (хранение в сессии), добавление/удаление товаров.
- `shop/` — настройки и корневые URL проекта.
- `PROJECT_SCHEMA.md` — схема проекта.
- В моделях/шаблонах добавлены **обучающие комментарии**, чтобы было проще разбирать логику.

## Цели обучения

- Понять, как устроены связи моделей (`Category` ↔ `Product`).
- Разобрать формирование URL через `reverse()` и `slug`.
- Изучить работу корзины через сессионное хранилище.
- Научиться читать и комментировать код, чтобы лучше понимать, как Django собирает страницы.

## Запуск (пример)

1. Активируй виртуальное окружение (если используется):
   - `source venv/bin/activate`
2. Установи фреймвор django:
   - `pip install django`
3. Примени миграции:
   - `python manage.py migrate`
4. Запусти сервер:
   - `python manage.py runserver`
